# Запуск приложения

1. Для начала нужно поменять URI базы данных в кофигурационном файле config.yaml
2. Далее нужно запустить сервер

### Смена конфигурации
1. Переходим в [configs/config.yaml](configs/config.yaml)
2. Меняем URI базы данных на свой

### Запуск сервера
1. Билдим сервер `go build -o todoApp ./cmd/main/main.go`
2. Запускаем сервер `./todoApp`
